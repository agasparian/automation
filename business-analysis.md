## 2. АНАЛИЗ ПРОЦЕССОВ: что делается вручную

### Этап 1 — Лидогенерация

| Действие | Кто | Время |
|---|---|---|
| Публикация постов в ТГ, ВКонтакте | Менеджер / SMM | 3–5 ч/нед |
| Мониторинг рейтинговых сайтов (отзывы, рейтинги) | Менеджер | 1–2 ч/нед |
| Обновление SEO-контента сайта | SEO-специалист | постоянно |

### Этап 2 — Первый контакт

| Действие | Кто | Время |
|---|---|---|
| Обработка входящих заявок из 5+ каналов (сайт, ТГ, ВК, WhatsApp, почта) | Менеджер | 1–2 ч/день |
| Квалификация лида: узнать нишу, задачи, бюджет | Менеджер по продажам | 15–30 мин/лид |
| Назначение времени созвона (перебор в переписке) | Менеджер | 10–20 мин/лид |

### Этап 3 — Брифинг

| Действие | Кто | Время |
|---|---|---|
| Отправка брифа клиенту (вручную) | Менеджер | 10–15 мин/клиент |
| Заполнение брифа по Zoom / телефону | Менеджер | 1–2 ч/клиент |
| Внесение данных из брифа в рабочие документы | Менеджер | 30–60 мин/клиент |

### Этап 4 — Подготовка КП

| Действие | Кто | Время |
|---|---|---|
| Анализ сайта и маркетинга клиента | Менеджер / Аналитик | 2–4 ч/клиент |
| Подбор услуг и расчёт стоимости | Руководитель | 1–2 ч/клиент |
| Оформление КП (презентация) | Менеджер / Дизайнер | 2–4 ч/клиент |

### Этап 5 — Защита КП

| Действие | Кто | Время |
|---|---|---|
| Проведение Zoom-встречи | Менеджер / Руководитель | 1–1,5 ч |
| Follow-up после встречи (письмо/сообщение с итогами) | Менеджер | 20–40 мин |
| Напоминания при задержке ответа клиента | Менеджер | 10–15 мин/лид |

### Этап 6 — Онбординг

| Действие | Кто | Время |
|---|---|---|
| Формирование договора по шаблону | Менеджер / Юрист | 30–60 мин |
| Выставление счёта | Бухгалтер / Менеджер | 15–30 мин |
| Сбор доступов (сайт, Директ, Метрика, соцсети) | Менеджер проекта | 1–3 ч |
| Постановка задач команде + введение в проект | Руководитель проекта | 1–2 ч |

### Этап 7 — Производство и отчётность

| Действие | Кто | Время |
|---|---|---|
| Написание медицинских текстов | 15 райтеров + 2 редактора + корректор | основной объём |
| Подготовка ежемесячного отчёта на каждого клиента | Менеджер + Специалисты | 3–5 ч/клиент |
| Рассылка отчётов клиентам (вручную) | Менеджер | 10–15 мин/клиент |
| Контроль дедлайнов по задачам | Менеджер проекта | 1–2 ч/день |
| Сбор обратной связи от клиентов | Менеджер | 15–30 мин/клиент/мес |

## 3. КАНДИДАТЫ НА АВТОМАТИЗАЦИЮ (топ-5)

### 🔴 #1 — Подготовка ежемесячных отчётов клиентам

- **Повторяемость:** ежемесячно, ~25 раз (по числу клиентов)
- **Трудозатраты:** 3–5 ч × 25 клиентов = **75–125 ч/месяц**
- **Почему шаблонно:** одинаковая структура — данные из Яндекс Метрики, Директа, Search Console, динамика позиций SEO, количество обращений. Только цифры меняются.
- **Боль:** специалисты тратят время на копирование данных вместо их анализа

### 🔴 #2 — Квалификация входящих лидов и обработка заявок

- **Повторяемость:** ежедневно, каждый новый лид
- **Трудозатраты:** 1–2 ч/день = **30–45 ч/месяц**
- **Почему шаблонно:** одни и те же вопросы — ниша, текущий сайт, бюджет, задача, контакт. Скрипт квалификации фиксирован.
- **Боль:** заявки приходят из 5+ каналов, менеджер переключается постоянно; лиды теряются

### 🟠 #3 — Онбординг нового клиента

- **Повторяемость:** каждый новый клиент (~2–3 в месяц при росте)
- **Трудозатраты:** 3–5 ч/клиент = **6–15 ч/месяц**
- **Почему шаблонно:** одинаковый чеклист — договор, счёт, сбор доступов, уведомление команды, создание папок/задач, первый бриф с командой
- **Боль:** из-за ручного онбординга теряются шаги, клиент ждёт, первое впечатление страдает

### 🟠 #4 — Подготовка коммерческого предложения

- **Повторяемость:** каждый новый лид (квалифицированный)
- **Трудозатраты:** 4–6 ч/КП
- **Почему шаблонно:** структура КП одинаковая — о компании, кейсы, набор услуг под задачу, цены, условия. Персонализация — данные из брифа.
- **Боль:** руководитель тратит время на рутинные расчёты вместо стратегии

### 🟡 #5 — Назначение созвонов и напоминания (лиды и клиенты)

- **Повторяемость:** ежедневно
- **Трудозатраты:** 30–60 мин/день = **15–25 ч/месяц**
- **Почему шаблонно:** стандартные сообщения — подтверждение встречи, напоминание за 1 час, follow-up через 2 дня, «не забыли ли вы о нас»
- **Боль:** менеджеры забывают делать follow-up, лиды остывают

## 4. ПРИОРИТЕТЫ АВТОМАТИЗАЦИИ

```

ВЫСОКИЙ ЭФФЕКТ
      │
 #1   │  #2
 ─────┼──────────────────────────────
 #3   │  #4   #5
      │
НИЗКИЙ ЭФФЕКТ
      └──────────────────────────────
      НИЗКОЕ УСИЛИЕ    ВЫСОКОЕ УСИЛИЕ
```

### Приоритет 1 — Автоматические ежемесячные отчёты ⭐⭐⭐⭐⭐

**Эффект:** экономия 75–125 ч/месяц → при ФОТ 150 ₽/ч = **11 000–18 000 ₽/месяц**
**Усилие:** среднее
**Инструменты:** n8n (у вас уже есть!) + API Яндекс Метрики + API Яндекс Директ + шаблон Google Slides/Docs
**Что делает автоматизация:**
- Каждый месяц n8n запрашивает данные из Метрики, Директа, Search Console
- Подставляет в шаблон отчёта
- Формирует PDF и отправляет клиенту в Телеграм/на почту
- Уведомляет менеджера для добавления комментариев

---

### Приоритет 2 — Квалификация лидов через бота ⭐⭐⭐⭐⭐

**Эффект:** экономия 30–45 ч/месяц + 0 потерянных лидов
**Усилие:** низкое (Телеграм-бот у вас уже есть)
**Инструменты:** n8n + Телеграм-бот + CRM/таблица
**Что делает автоматизация:**
- Бот принимает заявку из любого канала (через единый вход)
- Задаёт 5 квалификационных вопросов (ниша, сайт, бюджет, задача, контакт)
- Записывает данные в CRM / Google Sheets
- Назначает созвон через Calendly-ссылку
- Уведомляет менеджера с карточкой лида

---

### Приоритет 3 — Онбординг нового клиента ⭐⭐⭐⭐

**Эффект:** экономия 6–15 ч/месяц + улучшение первого впечатления клиента
**Усилие:** низкое
**Инструменты:** n8n + шаблонные сообщения + чеклист в Notion/Trello
**Что делает автоматизация:**
- По триггеру «договор подписан» запускает сценарий
- Отправляет клиенту приветственное письмо + запрос доступов по шаблону
- Создаёт задачи в таск-менеджере для каждого специалиста
- Через 3 дня проверяет получение доступов, напоминает

---

### Приоритет 4 — Напоминания и follow-up по лидам ⭐⭐⭐

**Эффект:** экономия 15–25 ч/месяц + рост конверсии из КП в продажу
**Усилие:** низкое
**Инструменты:** n8n + Телеграм-бот + CRM
**Что делает автоматизация:**
- После отправки КП автоматически ставит задачи: follow-up через 2 дня, звонок через 5 дней
- Отправляет менеджеру напоминание с контекстом лида
- Отправляет клиенту «мягкое» напоминание через бот, если нет ответа

---

### Приоритет 5 — Автоматическая сборка КП по брифу ⭐⭐⭐

**Эффект:** экономия 4–6 ч/КП → при 4 КП/месяц = 16–24 ч/месяц
**Усилие:** среднее-высокое
**Инструменты:** n8n + Claude API + шаблон КП в Google Slides
**Что делает автоматизация:**
- Данные из заполненного брифа → Claude API формирует персонализированный текст КП
- Подставляет в шаблон презентации нужные услуги и цены
- Отправляет черновик менеджеру на финальную проверку

## РЕКОМЕНДАЦИЯ ПО ПЕРВЫМ ШАГАМ

Поскольку у МедСпринт уже есть:
- ✅ n8n (создание нейроворонок — ваша услуга)
- ✅ Телеграм-бот
- ✅ Опыт с AI-инструментами

**Начать с #2 (квалификация лидов) — за 1–2 недели, инвестиции минимальны, эффект виден сразу.**
**Параллельно запустить #1 (отчёты) — это даст максимальную экономию времени команды и прямо влияет на маржинальность.**

Вместе эти два процесса высвобождают **100–170 часов в месяц** без найма новых сотрудников — прямой путь к удвоению маржинальности, которое стоит в целях на 2026 год.
