## ⚠️ ИЗМЕНЕНИЕ АРХИТЕКТУРЫ: нет Google Workspace

**Проблема:** без корпоративного домена Google Forms нельзя ограничить по домену.

**Решение: email-allowlist в Google Sheets + проверка в n8n**

```
[Google Form — открытая, но со сбором email]
        │
        ▼ новая строка
[n8n]
  └──► Проверить: email отправителя есть в листе «Допущенные»?
          │
          ├── НЕТ → отправить email «У вас нет доступа» → STOP
          │
          └── ДА → продолжить сценарий
```

**Лист «Допущенные» в Google Sheets:**
| Email | Имя | Роль |
|---|---|---|
| ivanov@gmail.com | Иванов А. | Менеджер |
| petrova@gmail.com | Петрова С. | Аккаунт |

Добавить нового сотрудника = добавить одну строку в таблицу. Удалить = удалить строку.
В форме обязательно включить «Собирать адреса электронной почты».

   - **Клиент** — тип «Раскрывающийся список», обязательное
   - **Услуга** — тип «Раскрывающийся список», обязательное

## ФАЗА 3 — n8n: логика получения данных клиента

**Исполнитель:** разработчик
**Зависит от:** фазы 2, завершения фазы 0.1

### Шаги:
1. *(только при Google Forms без Workspace)* Добавить ноду **Google Sheets** (Read):
   - Лист: «Допущенные»
   - Фильтр: `Email == {{form.email}}`
   - Ноду **IF**: email найден? Нет → **Send Email** «У вас нет доступа» → **STOP**

2. Добавить ноду **Google Sheets** (Read):
   - Лист: «Клиенты»
   - Фильтр: `Название клиента == {{form.Клиент}}`
   - Результат: ID папки, последний номер приложения, ID последнего Doc

3. Добавить ноду **IF** — клиент найден?
   - **Нет** → **Send Email** «Клиент не найден, проверьте написание» → **STOP**
   - **Да** → продолжить

4. Добавить ноду **Google Docs** (Get Document) — читаем предыдущее приложение:
   - Document ID: `{{клиент.ID_последнего_Doc}}`
   - Цель: извлечь номер и дату основного договора

5. Добавить ноду **Code** (JS) — извлечь из предыдущего приложения три значения:
   ```js
   // Собираем текст из структуры Google Docs API
   const text = $input.item.json.body.content
     .map(el => el.paragraph?.elements?.map(e => e.textRun?.content).join('') || '')
     .join('\n');

   // Номер и дата основного договора: «к Договору № X от Y г.»
   const matchDogovor = text.match(/к Договору №?\s*([\w\/\-]+)\s+от\s+([\d\w\s]+г\.)/);

   // Номер приложения: «Приложение № N» в начале документа
   const matchNomer = text.match(/Приложение\s*№\s*(\d+)/);

   return {
     nomerDogovora:    matchDogovor ? matchDogovor[1].trim() : '',
     dataDogovora:     matchDogovor ? matchDogovor[2].trim() : '',
     nomerPrilozhenia: matchNomer   ? String(parseInt(matchNomer[1]) + 1) : ''
   };
   ```

7. Добавить ноду **Code** (JS) — извлечь Document ID из URL реквизитов:
   ```js
   const url = $input.item.json['Ссылка на файл реквизитов клиента'];
   const match = url.match(/\/document\/d\/([a-zA-Z0-9_-]+)/);
   return { docId: match ? match[1] : null };
   ```

8. Добавить ноду **IF** — docId найден?
   - **Нет** → **Send Email** «Неверная ссылка на файл реквизитов» → **STOP**

9. Добавить ноду **Google Docs** (Get Document) — читаем файл реквизитов клиента:
   - Document ID: `{{docId}}`

10. Добавить ноду **Code** (JS) — извлечь поля из текста реквизитов:
    ```js
    const lines = text.split('\n');
    const fields = {};
    lines.forEach(line => {
      const [key, ...val] = line.split(':');
      if (key && val.length) fields[key.trim()] = val.join(':').trim();
    });
    return fields; // { 'Полное название': ..., 'ФИО подписанта': ... }
    ```

11. Добавить ноду **Google Sheets** (Read):
    - Лист: «Шаблоны»
    - Фильтр: `Услуга == {{form.Услуга}}`
    - Результат: ID шаблона на Google Drive

12. Добавить ноду **IF** — шаблон найден?
    - **Нет** → **Send Email** «Шаблон для услуги [X] не найден» → **STOP**
    - **Да** → продолжить

**Результат фазы 3:** n8n знает номер нового приложения, ID шаблона, все реквизиты клиента.

## ФАЗА 4 — n8n: создание документа

**Исполнитель:** разработчик
**Зависит от:** фазы 3, завершения фазы 0.2

### Шаги:
1. Добавить ноду **Google Drive** (Copy File):
   - Source File ID: `{{шаблон.ID}}`
   - Destination Folder ID: `{{клиент.ID_папки}}`
   - Новое имя: `Приложение №{{новый_номер}} — {{клиент.Название}}`
   - Результат: ID нового документа

2. Добавить ноду **Google Docs** (Get Document) — получить тело документа для проверки (опционально)

3. Одна нода **HTTP Request** → Google Docs API `batchUpdate` — заменяет все 6 плейсхолдеров за один вызов:
   ```json
   POST https://docs.googleapis.com/v1/documents/{{новый_doc_id}}:batchUpdate

   {
     "requests": [
       { "replaceAllText": { "containsText": { "text": "{{НОМЕР_ПРИЛОЖЕНИЯ}}" },
           "replaceText": "{{новый_номер}}" } },
       { "replaceAllText": { "containsText": { "text": "{{НОМЕР_ДОГОВОРА}}" },
           "replaceText": "{{клиент.Номер_договора}}" } },
       { "replaceAllText": { "containsText": { "text": "{{ДАТА_ДОГОВОРА}}" },
           "replaceText": "{{клиент.Дата_договора}}" } },
       { "replaceAllText": { "containsText": { "text": "{{ДАТА_ПРИЛОЖЕНИЯ}}" },
           "replaceText": "{{form.Дата_приложения}}" } },
       { "replaceAllText": { "containsText": { "text": "{{ДАТА_НАЧАЛА_РАБОТ}}" },
           "replaceText": "{{form.Дата_начала_работ}}" } },
       { "replaceAllText": { "containsText": { "text": "{{НАЗВАНИЕ_КЛИЕНТА}}" },
           "replaceText": "{{реквизиты.Полное_название}}" } },
       { "replaceAllText": { "containsText": { "text": "{{ФИО_ПОДПИСАНТА_КЛИЕНТА}}" },
           "replaceText": "{{реквизиты.ФИО_подписанта}}" } }
     ]
   }
   ```
   > Даты форматировать перед подстановкой: `01.04.2026` → `1 апреля 2026 г.` (нода Code перед HTTP Request)

**Результат фазы 4:** готовый документ лежит в папке клиента на Google Drive.

## ФАЗА 5 — n8n: обновление реестра и уведомление

**Зависит от:** фазы 4

### Шаги:
1. Добавить ноду **Google Sheets** (Append Row):
   - Лист: «Реестр приложений»
   - Данные: дата, клиент, услуга, новый номер, ссылка на Doc (`https://docs.google.com/document/d/{{doc.ID}}`), статус «Черновик», email отправителя формы

3. Добавить ноду **Send Email** (или Telegram):
   ```
   Кому:   {{form.email_отправителя}}
   Тема:   Готово: Приложение №{{новый_номер}} — {{клиент.Название}}
   Тело:
   Приложение создано автоматически. Проверьте и отправьте клиенту.

   Документ: https://docs.google.com/document/d/{{doc.ID}}
   Клиент:   {{клиент.Название}}
   Услуга:   {{form.Услуга}}
   ```
